About The Project

This project was for my MIS15 class completed on October, 2016. For this project we had to draw an animal using Visual Basic's Windows Form application. The language that we used was C#, I chose to draw a dolphin.

Requirements & Action

The requirements that were given to us were that the finished product had to contain at least 50 lines of code. The actions that I took was create the program that displayed a drawing of a dolphin when completed. This was all done with the help of a coordinate chart that helped me plot the points on the chart to draw lines, polygons, and ellipses to draw the dolphin.

What I Learned

From this project I learned how to assemble one large product by putting together smaller bits of code. This also helped me stay consistent with my code all throughout the project. One of the more important things I learned was working alongside a deadline.
